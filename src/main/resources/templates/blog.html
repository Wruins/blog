少女意外穿越时空

p { color: red }
大家好，我是雷霆嘎丶巴。今天我们来聊一聊一部06年的剧场版动画《穿越时空的少女》，同《红辣椒》一样，这部动画电影也是改编自桐井康隆的同名小说，不过电影描述的是以小说剧情发生20年之后的原创故事，主角也由芳山和子换为她的侄女绀野真琴。

1,

影片对爱情的部分轻描淡写，其中心思想就是一句话——Time waits for no one！

影片的女主是一个有些笨手笨脚，不愿意长大的高二女学生。她的三大被动技能是：闯祸、躺枪以及逃避现实，有一天忽然获得了时间跳跃的能力，这个能力使她能够穿越到这些倒霉事情发生以前重新来过。说简单点就是一个事先知道了剧本的女孩，想让剧本里相关的人都能远离不幸，逃避现实。但是矛盾也出在这里，不幸最后不会被抹去，只会从一个人的身上转嫁到另一个人的身上。现实也终究要面对，由于女主的穿越行为而产生的蝴蝶效应，终究会作用到自己的身上，想通过跳跃时空来逃避问题和不幸，结果注定是逃不掉的，所以印证了影片的中心思想：Time waits for no one！

2,

女主不断穿越时空的行为，其实就是在身体年龄不变的情况下达到心理年龄日趋成年化

道理口头上说说，谁不会说；说出来了之后又有谁听不懂呢？我们经常能看到周遭一些明知理亏的人，依然蛮不讲理地胡搅蛮缠，就像青春期叛逆的少男少女一样，你说他不懂吗，你说他不知晓他人对自己的一片心意吗，你说他难道感觉不到在意他的人对他的关怀吗？都不是，当你看到那些不把自己好心的说教当作一回事的人的时候，你也许心中忿忿不平。不过身边要是有年老的长辈，也许会对你说：人生就是弯弯绕，你不让他去绕一遭的话，他永远也体会不到什么叫做刻骨铭心！而影片中穿越时空的能力让女主能够重复一个既定时段并不停地累加弯弯绕出来的那种刻骨铭心的宝贵经历，就像《哈利·波特》里得到了麦格教授的时间转换器的赫敏·格兰杰，可以回到几个小时前上更多的课程。

3,

通过能看见未来剧本的女主的视角，来达到对观众进行深刻说教Time waits for no one这个既是影片想要传达的核心思想，也是人们老生常谈的道理

我们仔细来回顾一下女主获得了穿越时空的能力之后，她所做的一切吧。她一开始所做的就是用能力让自己远离哪怕是微不足道的不幸——学校里面出的丑，文理分班必须选择时的纠结，以及对未来自己的发展规划感到迷惘，还有不得不为了上学而早起的无奈。她可以从中不断地试错，于是膨胀到认为自己能从无限可能性的未来中选出对自己最有利的那一种可能性。

再到她所逃避的这些不幸被原封不动地转嫁到他人身上时，她开始用自己的能力来让这些因自己而不幸的人远离本该归属她的不幸。最后从逃避来自好朋友的爱情表白，到真正意识到自己内心的真正感情，不再逃避、不再犹豫、倾尽全力去抓住它时，女主终于在绕了一个大弯之后，变得成熟起来，懂得了Time waits for no one这个道理——珍惜当下、不要逃避、直面现实、尽量做出不让自己后悔的选择，因为即使是获得穿越时空的能力，其使用次数也是有限的啊！让观众切身体会到女主的心路历程，体会到那种恍如隔世、抱憾终身所带给自己的刻骨铭心。

4,

时光不会倒流，过去不会改变，人生不会重来；你永远无法穿越时空重新来过，那么就请珍惜当下吧！

某种意义上，《穿越时空的少女》这部动画和《ReLIFE 重返17岁》有着异曲同工之妙。在《穿越时空的少女》中，女主通过不断的时空跳跃，累积经验，变得成熟，最后终于正视并表达了自己的感情；而在《ReLIFE 重返17岁》中，男女主通过从职场人士重回17岁校园生活一年的经历，用成熟的心智回到过去找到自己的初心，并深深地爱上彼此，矢志不渝。虽然这两部作品的主人公的成熟历程是反着来的，但他们本质上却是殊途同归的，那就是——时不我待、真诚相对；找回初心、莫留遗憾！
